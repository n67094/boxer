# Keyboard Input Handling

`bxr_key_e` values are based on scancode (the physical location of the
key on the keyboard), not keycode (the character generated by the key press,
which can change based on keyboard layout and modifiers).

To get the key value of the current keyboard layout, use `bxr_key_name` or
`bxr_key_name_ex` with the appropriate modifiers.

## Public API

<a name="line-198"></a>

```cpp
const char *bxr_key_name(bxr_key_e key)
```

Get the name of a key based on its scancode and the current modifier state.

Use the current modifier state to determine the key name.

`key` is the scancode of the key to get the name of.

`return` the name of the key, or "Unknown" if the key is not recognized.
